ajaxGet

DotUtils.ajaxGet = function(url,
params,
onComplete)

Static, Sends a simple asynchronous GET request

Parameters

url{String} The URL to send the request to
params{object} An object containing a list of parameters that should be attached to the url before sending the request.  This parameter is optionnal (can be undefined or null)
onComplete{Method} A callback function that will be called when the request completes.  It will be called with a string parameter containing the raw response’s body.
