Dot

Main entry point of Dot.js

Instances of this class can be used to place and configure Dot graphs on the page

setScale

setScale: function(scale)

Sets the display scale of the graph

Parameters

scale{float} The new scale to apply.  1 is original size.

setImagePath

setImagePath: function(imagePath)

Sets the base path where images specified by XDot files will be searched for.

Parameters

imagePath{String} The base directory containing the graph images.

load

load: function(url,
urlParams)

Loads a new XDot source for the graph

Note that this method will automatically parse the XDot file and update the graph.

Parameters

url{String} The XDot file URL
urlParams{object} An optionnal object containing key/value pairs that will added as URL parameters before the XDot request is sent

parse

parse: function(xdot)

Parses the provided XDot instructions and updates the graph according to them

Parameters

xdot{String} The contents of an XDot file

draw

draw: function(redrawCanvasOnly)

Updates the graph graphics

This method is generally triggered from other methods, so you should not have to call it explicitely

Parameters

redrawCanvasOnly{bool} Optionnal.  Defines if the method should only redraw the canvas contents or if the entire DOM tree should be updated.  Default is false.

ajaxGet

Dot.ajaxGet = function(url,
params,
onComplete)

Sends a simple asynchronous GET request

Parameters

url{String} The URL to send the request to
params{object} An object containing a list of parameters that should be attached to the url before sending the request.  This parameter is optionnal (can be undefined or null)
onComplete{Function} A callback function that will be called when the request completes.  It will be called with a string parameter containing the raw response’s body.
